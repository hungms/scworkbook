# Single Cell Workbook (scworkbook)

1. Data Processing
2. Cell Classification
3. Processing CITEseq
4. Processing VDJseq
5. Infer Trajectory
6. Predict Gene Network
7. Predict Cell Communication
8. Bulk Deconvolution
